A plagiarism detection web application that checks the internet for similarities in English and Spanish.

Written with Flask, Python-based micro web framework.
Using Python's SQLAlchemy as a database.
Using Copyleaks API to detect similarities across the internet and return matches.
Using Microsoft Azure API to translate to Spanish.

Page links that exist:
http://snitched.me/
http://snitched.me/login
http://snitched.me/create
http://snitched.me/userpage

Functionality:
- Create account
- User login/maintain user session
- Logout
- Submit text
- Check for matches in english
- Translate to spanish and check for matches
- Stores all user's submitted documents
- View each submitted document
